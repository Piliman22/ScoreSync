# ScoreSync
## Usage
```bash
git clone https://github.com/Piliman22/ScoreSync
```
```bash
cd ScoreSync
```
```bash
npm install
```
Create a folder in the levels file with any name you like and put usc in it.

project-root/
├── README.md
├── src/
├──lib  
└── levels/
    └── your_folder_name/
        └── chart.usc
            cover.png
            bgm.mp3


```bash
npm run start
```
